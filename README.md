# Homepage of Infinite Magazine

## Structure of the site

### Pages

This folder contains the root files for the pages of the website: About, Home, IssueView, and PastIssues

### Components

This folder contains react elements that are reused across pages

### CSS

This folder contains the CSS files that describe the layout and style of the website

## How to deploy

In order to deploy any changes made to the site execute ```npm run build```
followed by ```npx gh-pages -d build```
